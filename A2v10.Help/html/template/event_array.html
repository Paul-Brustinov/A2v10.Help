События массивов

События

Описание

События массивов:

  • .adding - перед добавлением элемента
  • .add - элемент добавлен
  • .remove - элемент удален
  • .select - элемент выбран

Прототип

function(arr, elem)

Параметры

  • this - корень модели данных (TRoot)
  • arr - изменяемый массив
  • elem - новый или удаленный элемент

Возвращает

Имеет смысл только для события .adding.

  • Boolean - строгое значение false. Отменяет операцию добавления элемента в массив.
