Модель на клиенте

Свойства объекта

Каждый объект расширяется следующимии служебными свойствами и методами:

{
  // свойства
  $root: Object,
  $parent: Object,
  $ctrl: Object,
  $id: Object,
  $name: Object,
  $isNew: Boolean,
  $isEmpty: Boolean,
  // методы
  $empty: Function,   // очищает объект
  $merge: Function
}

Дополнительные свойства корневого объекта

Корневой объект (TRoot) кроме основных свойств, расширяется следующимии дополнительными служебными свойствами и методами:

{
  // свойства
  $template: Object,
  // методы
  $setDirty: Function
  $defer: Function,
  $forceValidate: Function
}

Свойства массива

Каждый массив расширяется следующимии дополнительными служебными свойствами и методами:

{
  // свойства
  $root: Object,
  $parent: Object,
  $ctrl: Object,
  $isEmpty: Boolean,
  // методы
  $empty: Function,   // очищает объект
  $remove: Function,
  $select: Function,
  $clearSelected: Function,
  $append: Function,
  $prepend: Function,
  $insert: Function
  $copy: Function,
}
