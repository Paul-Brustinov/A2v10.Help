Простые элементы

Простые элементы разметки, как правило, поддерживают конвертирование из строки. Это позволяет значительно сократить код разметки.

Простые элементы

  • Length - длина
  • Size - размер
  • Thickness - размер прямоугольной области

Коллекции

  • UIElementCollection - коллекция элементов
