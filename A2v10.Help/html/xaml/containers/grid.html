Элемент Grid

Все контейнеры

Представляет собой сетку из строк и колонок.

Примечание. Контейнер Grid используется в макетах пользовательского интерфейса. Он не предназначен для отображения данных. Для отображения и привязки к данным используйте соответствующие элементы упрвления, такие как Table или DataGrid.

Свойства

Имя Тип Описание
Children UIElementCollection Свойство содержимого. Коллекция дочерних элементов сетки. Унаследовано от Contrainer.
If Boolean? Включать ли элемент управления в поток рендеринга.
Height Length Высота элемента управления
Rows RowDefinitions Определения строк сетки
Columns ColumnDefinitions Определения колонок сетки

Присоединенные свойства

Имя Тип Описание
Row Int32 Определяет номер строки, в которую должен быть помещен дочерний элемент
Col Int32 Определяет номер колонки, в которую должен быть помещен дочерний элемент
RowSpan Int32 Определяет количество строк, которые будет занимать дочерний элемент. По умолчанию - 1.
ColSpan Int32 Определяет количество колонок, которые будет занимать дочерний элемент. По умолчанию - 1.
VAlign VerticalAlign Определяет вертикальное выравнивание дочернего элемента таблицы. По умолчанию - Top.
