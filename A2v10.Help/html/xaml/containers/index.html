Контейнеры

Контейнерные элементы или просто контейнеры используются для размещения внутри себя других элементов.

Контейнеры

  • Block - блок
  • Dialog - диалог
  • DropDownMenu - выпадающее меню
  • Grid - сетка
  • GridGroup - группа элементов сетки
  • Include - вложенное содержимое
  • Partial - часть страницы
  • Splitter - разделитель
  • StackPanel - панель
  • Taskpad - панель задач
