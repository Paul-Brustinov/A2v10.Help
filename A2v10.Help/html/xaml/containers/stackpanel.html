Элемент StackPanel

Все контейнеры

Представляет собой контейнер, в котором дочерние элементы выравниваются в одну линию, ориентированную вертикально или горизонтально.

Базовый элемент: Container

Свойства

Имя Тип Описание
Orientation Enum Ориентация панели. Возможные значения:
  • Vertical - элементы размещаются сверху вниз
  • Horizontal - элементы размещаются слева направо
AlignItems Enum Выравнивание дочерних элементов. Возможные значения:
  • Default -
  • Baseline -
  • Center -
  • Start -
