Элемент FullHeightPanel

Все контейнеры

Представляет панель, высота которой всегда равна 100%.

Наследует: Container : UIElement : UIElementBase

Свойства

Имя Тип Описание
MinWidth Length Минимальная ширина панели
Свойства базового элемента

Присоединенные свойства

Имя Тип Описание
FullHeightPanel.Fill Boolean Определяет дочерний элемент, высота которого будет изменяться при изменении высоты родительского контейнера.
Только один дочерний элемент может иметь свойство Fill = True.
