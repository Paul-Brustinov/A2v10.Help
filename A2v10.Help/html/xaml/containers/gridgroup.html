Элемент GridGroup

Все контейнеры

Представляет собой группу элементов внутри сетки Grid. Элемент не имеет внешнего представления и предназначен для того, чтобы было удобно скрывать/показывать группы элементов с помощью свойств If, Show, Hide.

Элемент GridGroup является "прозрачным" с точки зрения присоединенных свойств. Это означает, что присоединенные свойства (такие как, например Grid.Row, Grid.Col) дочерних элементов будут относиться не к GridGroup, а к родительскому контейнеру Grid

Свойства

Имя Тип Описание
Children UIElementCollection Свойство содержимого. Коллекция дочерних элементов сетки. Унаследовано от Contrainer.
If Boolean? Включать группу элементов в поток рендеринга.
Show Boolean? Показывать группу элементв
Hide Boolean? Скрывать группу элементв
