Элемент TagLabel

Все текстовые элементы

Представляет собой текстовую метку-маркер. Чаще всего используется для отображения состояния чего-либо.

Базовый элемент: Inline

Свойства

Имя Тип Описание
Content String Свойство содержимого. Текстовое содержимое метки.
Style Enum Стиль отображения метки. Возможные значения:
Default Green Success Warning Orange Info Cyan
Error Danger Red Purple Pink Gold Blue
Salmon Seagreen Tan Magenta LightGray
Свойства базового элемента
