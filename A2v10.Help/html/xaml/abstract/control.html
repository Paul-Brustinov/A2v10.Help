Абстрактный элемент Control

Все абстрактные элементы

Представляет собой абстрактный элемент управления.

Наследует: UIElement : UIElementBase

Производные элементы: ValuedControl, ContentControl, List, DataGrid, TreeView, Table

Свойства

Имя Тип Описание
Block Boolean Выводить элемент управления в виде блочного, а не строчного элемента.
Label String Текстовая метка для элемента управления. Обычно выводится сверху или слева от поля ввода.
Description String Описание для для элемента управления. Обычно выводится под полем ввода.
Required Boolean Обязательное значение для ввода. Обратите внимание, что это только визуальное представление обязательного ввода в поле ввода (конкретный вид зависит от темы пользовательского интерфейса). Для управления проверками используйте валидаторы шаблонов.
Disabled Boolean Элемент управления недоступен для ввода. Для управления этим полем обычно используется привязка Bind.
Width Length Ширина элемента управления. По умолчанию элемент занимает 100% ширины родительского элемента. Для некоторых контейнеров (например Toolbar) ширина вложенных элементов будет установлена в некоторое значение по умолчанию.
TabIndex Int32 Порядковый номер элемента в цепочке переходов по клавише Tab. Значение "-1" исключает элемент из цепочки переходов.
TestId String Идентификатор элемента для инструментов тестирования.
Popover Popover Всплывающая подсказка для элемента. Выводится в виде ссылки справа от элемента.
Hint Popover Всплывающая подсказка для элемента. Выводится в виде после метки.
Validator Validator Позволяет задать свойства валидатора элемента, такие как положение или ширину.
AddOns UIElementCollection Дополнительные элементы управления. Обычно это гиперссылки, которые выполняют какие-то дополнительные команды
