Элемент DataGridColumn

Все элементы управления

Представляет собой одну колонку таблицы DataGrid.

Свойство содержимого: Content

Свойства

Имя Тип Описание
Content Bind Свойство содержимого. Только привязка (Bind). Содержимое ячейки таблицы.
Header String Заголовок колонки.
Width Length Ширина колонки.
Fit Boolean Устанавливать ширину колонки по содержимому.
Wrap WrapMode Режим переноса для содержимого ячейки и для заголовка.
Align TextAlign Горизонтальное выравнивание содержимого ячейки.
Bold Boolean Отображать содержимое ячейки полужирным шрифтом
Small Boolean Отображать содержимое ячейки уменьшенным шрифтом.
Mark Object Только привязка (Bind). Маркировка ячейки таблицы.
Icon Icon Иконка, которая будет отображаться в таблице перед содержимым.
Editable Boolean Разрешено ли редактирование ячейки.
ControlType ColumnControlType Тип элемента управленя в ячейке таблицы.
Sort Boolean? Разрешена ли сортировка в данной колонке. Позволяет индивидуально переопределить сортировку по каждой колонке. Общее разрешение сортировки задается свойством Sort всей таблицы DataGrid.
SortProperty String Имя свойства в источнике данных, по которому будет выполняться сортировка. Если не указано, будет выполняться сортировка по свойству Content.
If Boolean? Включать колонку в поток рендеринга. Для того, чтобы показывать/скрывать колонки в зависмости от условий используется привязка (Bind).
Command BindCmd Команда для ячейки таблицы. Если команда указана, то содержимое (Content) будет выводится в виде гиперссылки со связанной командой.
