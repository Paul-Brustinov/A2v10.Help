Элемент TextBox

Все элементы управления

Представляет собой поле для ввода текста.

Наследует: ValuedControl : Control : UIElement : UIElementBase

Свойства

Имя Тип Описание
Multiline Boolean Многострочный элемент.
AutoSize Boolean Автоматически устанавливать высоту элемента в зависимости от содержимого. Только для многострочных элементов.
Rows Int32? Количество строк. Только для многострочных элементов.
Align TextAlign Выравнивание текста
Password Boolean Поле ввода пароля. Вместо текста отображаются заполнители. Копирование текста из поля невозможно.
Placeholder String Подсказка. Отображается в пустом поле ввода. Исчезает при получении полем фокуса ввода.
SpellCheck Boolean? Проверять ли орфографические ошибки. По умолчанию - зависит от настроек браузера.
UpdateTrigger UpdateTrigger Способ обновления значения модели данных, связанных с этим элементом.
