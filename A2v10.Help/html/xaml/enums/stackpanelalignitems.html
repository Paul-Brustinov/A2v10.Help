Перечисление StackPanelAlignItems

Все перечисления

Задает выравнивание элементов для StackPanel

Значения

Значение Описание
Default По умолчанию. Не установлено
Baseline По базовой линии
Center По центру
Start По началу оси (влево для горизонтальных панелей, вверх для вертикальных)
End по концу оси (вправо для горизонтальных панелей, вниз для вертикальных)
